# Rachit Kakkad

Hi, I'm Rachit — a full-stack developer and AI engineer from Gandhinagar, Gujarat, India, currently pursuing B.Tech in Computer Science (Class of 2029).

I build things across the stack: production React/TypeScript frontends, Django/Node.js backends, AI-powered systems, and blockchain tooling. I'm the founder of **COS (Cognitive Operating System)** — a personal AI assistant framework — and a core contributor to **WorkSense**, a B2B enterprise workforce intelligence platform built by Team MindMesh.

Some things I've shipped:
- **JARVIS** — a real-life offline AI voice assistant in Python (Claude API, Whisper, wake-word detection, full system control)
- **COS** — a cognitive AI layer for personal productivity and decision-making
- **ThreatLens** — real-time security intelligence dashboard
- **LifeLens AI** — wellness intelligence platform
- **AgriCert** — blockchain-based agricultural certification system
- **CodingGita Auction Engine** — real-time bidding platform

I'm a regular hackathon builder, and I've worked on everything from Odoo ERP integrations to fleet management systems — which means I'm comfortable diving into large, opinionated codebases.

## Why Wagtail?

I've been thinking about CMS design for a while. Wagtail is one of the few CMSes where the admin UI is genuinely considered part of the product — not an afterthought. The fact that NASA, Mozilla, and the NHS trust it tells you something about the quality bar the community holds itself to.

I want to contribute meaningfully to that — not just fix typos, but understand the architecture, improve real user-facing experiences, and write code that belongs in a project this mature.
