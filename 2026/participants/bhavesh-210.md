## Introduction

Hello everyone, I'm Bhavesh Sharma, and I'm excited to be a part of this community and very eager to contribute to Wagtail for GSoC.

I'm interested in the **Starter Kit** project. The Wagtail Starter Kit is a pre-built news website template that helps developers quickly get started with Wagtail CMS. Right now it's outdated and hard to maintain. This GSoC project aims to modernize it with the latest Wagtail features and add automation to keep it updated — making it the best starting point for anyone new to Wagtail.
